Game: 2048

How to play: 
Goal for this game is to reach the score.
Press Normal, Hard or Extreme to set the goal to 2048, 4096 and 8192 respectively.

Check Helper button to see the rules!
Use 'w, a, s, d' to move the tiles. Tiles with the same number will merge into one when they collide.
Press 'w' to slide all tiles up.
Press 'a' to slide all tiles left.
Press 's' to slide all tiles down.
Press 'd' to slide all tiles right.
